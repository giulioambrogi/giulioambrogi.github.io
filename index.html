Master Thesis Project

Description

Development of web applications to download 24/7, store and analyze mobility data retrieved from Twitter within the metropolitan area of Turin.

Key web application:

  • Tweet Downloader: download, store and visualize tweets in real-time
  • Mobility Data Analyzer: visualize data, user positions and usage areas of the city

Development

Abstract

The big growth of social networks during the last decade revolutionized the way human beings interact and led companies to introduce new business strategies. The big amount of data generated every day on the most common social platforms can be valuable for many fields of study and also for mobility. Sure enough, by processing correctly those data it's possible to extract informations about urban mobility, traffic flows and, more in general any kind of information that can help in developing new strategies. The aim of this work is to develop software instruments supporting the download and the analysis of data retrieved by Twitter within the metropolitan area of Turin. TweetDowloader and MobilityDataAnalzyer are the most relevant web application produced within this work: the first one is meant to download tweets 24/7 and store it into a database, the second one was made to visualize and analyze the downloaded data. In particular with MobilityDataAnalyzer the user can visualize general statistics, search data, get the positioning of the most active users and label data. Those web application ara designed to be scalable, able to handle big amount of data, well structured in order to make the code more readable, editable and extensible in the future, and to display good looking and responsive interfaces. In order to fulfill those goals the software was implemented with some of the most used frameworks and languages in enterprise environment: Spring Framework, AngularJS, Twitter Bootstrap, JavaEE, HTML 5, Javascript e CSS3.

IPsec Plugin

Description

This plugin translates high-level security policies to low-level IPsec system configurations, in order to allow users without technical knowledge to define security policies for its devices.

This project has been done in collaboration with TORSEC research group (Polytechnic University of Turin) within the course of Internet Security (Prof. Antonio Lioy) and it is part of the FP7 programme committed by European Commission.

Queue Assist simulation

Description

Queue Assist, also known as Adaptive Cruise Control, helps the driver to maintain an even speed combined with a pre-selected time interval to the vehicle ahead.

In this simulation I used a Raspberry Pi and a HC-SRO4 distance sensor in order to simulate the car network and the micro controller.

The main goal of this project is to build robust software respecting ISO 26262 standard.

This project has been done within Model Based Software Development course at Polytechnic University of Turin.
