Why this blog?

This blog started as a bunch of personal notes and ended up being published online. My aim is to show and explain small topics/issues I came across in a concise way, in particular I would like to focus on topics that are often misunderstood or underrated by developers, or maybe by me.

I normally work with both back-end and front-end technologies so you may find articles regarding different software development areas (and not only).

Why background-color:red?

If you’ve ever dealt with CSS code you might have already already got it.
In case you haven’t - long story short - background-color:red is a bit of CSS code that is often used on an element in order to highlight its position/shape on the screen - AKA debugging technique for impatient developers.
Disclaimer: that’s not exactly the best way of debugging.
